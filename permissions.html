File Permissions

  1. Next: File Linking to Smart Phone
  2. Previous: RF Codes
  3. File Permissions
Certain files need to have writing and executable privileges to work properly. The easiest way to enable this is to select these files (as mentioned below) in a SFTP client like WinSCP, right-click them and select properties, and assign the Owner writing and executable priviledges by clicking on the [ ]W and [ ]X boxes, respectively. You can also accomplish this via the command-line using the command: "chmod 744 filename.ext"

Files requiring writing/excutable (744) privileges:
alarm.py
activate_system.sh
automate_tasks.txt
cleanup.py
deactivate_system.sh
motion
motion-mmalcam.conf
on_file1.sh
on_file_new.py
on_file_new1.py
on_file_new2.py
reboot.py
remove_push.py
restart_insync.py
shutdown.py
vacation_away.py
vacation_return.py
